# Курсовая по NLP на Python

### Требования

xelatex, python3

### Настройка

```
pip install -r requirements.txt
```
открыть в Jupyter файл *main.ipynb*
при первом запуске раскоментить строки
```python
# import nltk
# nltk.download('punkt')
```
потом закоменнтить обратно.

# Использование

1. Подать входной текст в *input.txt*
2. Прогнать весь файл *main.ipynb* Jupyter Notebook
3. Собрать latex document: ```xelatex main```
